
Este proyecto reúne el trabajo colaborativo del grupo en el análisis del conjunto de datos del Titanic. El objetivo es explorar la información usando Pandas, comprender sus variables, crear nuevas características, comparar los diferentes archivos y generar visualizaciones que ayuden a interpretar mejor los datos y los patrones de supervivencia. A lo largo del proceso se documentan los hallazgos, las transformaciones realizadas y las conclusiones obtenidas.

El repositorio mantiene la estructura solicitada en el taller, incluyendo notebooks, scripts, resultados, documentación y control de versiones mediante ramas y Pull Requests, lo que permite mostrar de forma clara y ordenada el desarrollo completo del análisis.

No subir archivos CSV ni datasets grandes a esta carpeta.  
Aquí van solamente los datos (locales) descargados de Kaggle.

Estructura sugerida:
- data/raw/        
- data/processed/  

Instrucciones:
1. Descarga los datasets de Kaggle.
2. Colócalos dentro de la carpeta data/raw en tu desktop local.
3. Mantén el archivo README.md en el repositorio, pero no subas CSVs.
